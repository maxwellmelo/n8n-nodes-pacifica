# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-12-29

### Added
- Initial release
- **Market Data Operations**
  - Get Market Info
  - Get Prices
  - Get Orderbook
  - Get Candles (OHLCV)
  - Get Recent Trades
  - Get Historical Funding
- **Account Operations**
  - Get Account Info
  - Get Trade History
- **Order Operations**
  - Create Market Order
  - Create Limit Order
  - Cancel Order
  - Cancel All Orders
  - Get Open Orders
- **Position Operations**
  - Get Positions
  - Update Leverage
  - Update Margin Mode
  - Close Position
- **Subaccount Operations**
  - Create Subaccount
  - List Subaccounts
  - Transfer Funds
- Mainnet and Testnet support
- Agent Wallet authentication (secure API trading without withdrawal access)
- Full TypeScript implementation
